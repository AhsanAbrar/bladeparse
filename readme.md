A New `Markdown` parser for Laravel built on [Parsedown](https://github.com/erusev/parsedown) and [Parsedown Extra](https://github.com/erusev/parsedown-extra).

## Installation

```shell
composer require ahsanabrar/bladeparse
```


## Usage

```xml
@markdown('# Heading')

or

@markdown($markdown)

or

@markdown
# Heading
## Other Heading
@endmarkdown
```

That's it.
